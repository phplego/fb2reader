📖 FB2 → Translator
Upload a .fb2 file — all paragraphs will be numbered. Clicking a paragraph sends a request to OpenRouter for translation and shows the result below the original. The first translation will ask for an API key and store it in localStorage. Translations are cached locally (per file/language/paragraph).

🔑 OpenRouter API Key

Create a key at openrouter.ai/keys and paste it here. The key will be saved locally and used for translations.

Your key is stored only in your browser. You can remove it anytime via the “OpenRouter Key” button.

🎤 TTS Settings (OpenAI)

Configure OpenAI Text-to-Speech. Your settings are stored locally.

Tip: voices like sage, verse, or others may be available depending on the model.
